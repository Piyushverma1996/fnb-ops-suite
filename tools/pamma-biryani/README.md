# 📸 Pamma Biryani — Brand Site + Instagram Automation

A brand storefront and Instagram-content automation app for **Pamma Biryani**, a Hyderabadi biryani QSR.

**Live:** [pamma-biryani.vercel.app](https://pamma-biryani.vercel.app)

## What's inside

- **Brand storefront** — menu, story, location, order links
- **Instagram automation** — content scheduling, asset generation, caption helpers
